# node Installaiton Steps
## Step 1: Install node version manager (nvm) by typing the following at the command line
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
## Step 2: Load nvm by typing the following at the command line.
    source ~/.bashrc
## Step 3: Use nvm to install the latest LTS version of Node.js by typing the following at the command line.
    nvm install --lts
## Step 4: Test that Node.js is installed and running correctly by typing the following at the command line.
    node -e "console.log('Running Node.js ' + process.version)"
